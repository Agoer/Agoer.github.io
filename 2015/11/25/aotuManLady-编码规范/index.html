AotumanLady 编程规范

特别注意事项：

  1. 不使用 Swift 与 OC 混合编程。

  2. 不使用 StoryBoard 和 Xib 进行开发。（LaunchScreen 启动文件除外）

64 位支持

  • 优先使用 NSInteger，而不是 int、 long。

  • format 字符串中，禁止使用 %zd %tu。

  • 使用 %ld+ (long) 类型转换 或者使用 %@ 形式，以 @() 转换成 NSNumber 类型。

更多 64 位支持，阅读博客：64 位支持

命名

  • 禁止使用拼音，实在搞不定的英文单词可以查字典。

  • 遵循驼峰式命名规则。

  • 点击事件，以 didTapxxx、didClickxxx 形式命名。

  • 命名要有意义，禁止 int i，a，b 这种形式命名。

  • 使用 Modern ObjectiveC 语义。Modern Objective-C 详情

  • 使用字面量，如@(YES) @42 @”abc” @[ @1, @2, @3 ] @{ @”key1” : @”value1” } 。

  • 访问容器类成员尽量使用新的语法，如 dictionary[@”key”] array[0]

  • 构造函数返回类型为 instancetype

  • 枚举使用 NS_ENUM 或者 NS_OPTIONS

花括号

  • 函数、方法、语句块的花括号独占一行，其他的左花括号跟在前一个语法单元后面，中间加一个空格。

  • if 和 while 后面必须有花括号，即使只有一个空语句。

缩写

  • 尽量不使用缩写，对于众所周知的缩写，必须全部大写，比如 ID，URL，HTTP，对于不是众所周知的缩写，需要在项目中维护一个词汇表。

类型转换

  • 类型转换必须有注释。

特殊关键字

  • 除开特殊情况，不得使用 goto。特殊情况是指项目 leader 发起投票，多数项目成员同意。

注释

  • 不得嵌套注释。

  • 不得用注释禁用代码块。

  • 注释与对应的代码应该有相同缩进。

  • 接口封装应该在注释中标明接口文档地址。

  • TODO后面必须加上能表明作者的信息，比如 TODO LC … 。

  • 没有用的代码删掉，不要注释。

空白符

  • 关键字 if、else、while、for、switch 和 return，后面都必须有一个空格。

  • 赋值操作符 =、+=、-=、*=、/=、%=、&=、|=、^=、~= 和 !=，前后都必须有一个空格。

  • 二元操作符 +、-、*、/、%、<、<=、>、>=、==、!=、<<、>>、&、|、^、&& 和 ||，前后都必须有一个空格。

  • 一元操作符 +、-、++、–、!和 ~，操作符和操作数相邻一侧不得加空格，但操作符另一侧必须有一个空格。

  • 指针操作符 * 与类型之间必须有一个空格，与变量之间不得有空格。

  • 取地址操作符 & 与变量之间不得有空格。

  • 组成三元操作符的字符 ? 和 :，两侧必须有一个空格。

  • 结构指针和成员操作符 -> 和 .，两侧不得有空格。

  • 数组角标的左右括号 [ 和 ]，两侧不得有空格。

  • 含有小括号的表达式，与左右小括号相邻处不得有空格。

  • 函数名和左括号之间不得有空格。

  • 方法前面的 - 和 + 与返回值类型之间必须有一个空格，返回值类型与方法名之间不得有空格，方法签名中冒号两边不得有空格，除非冒号前面没有方法签名。

  • 每个逗号分隔的函数 / 方法参数后面必须有一个空格。

  • 每个用于分隔语句元素的分号后面加一个空格，每个用于结束语句的分号前面不得有空格。

  • 每份代码文件在结尾必须有一个空行。

  • 注意不允许提交只有一个空格或者空行的 pr 。

函数和方法

  • 私有函数用 static 标明，私有方法在最后加下划线。

  • 方法和函数名应该是表示动作的单词或短语。

  • 指定初始化方法必须用编译器指令标出 NS_DESIGNATED_INITIALIZER 。

  • 避免带有副作用的 getter 方法。

表达式和语句

  • 每个 case 中的 break，必须对齐关键字 case，而不是 case 的内容代码块。

类

  • 类成员变量必须以下划线开头。

  • @property 后面必须有一个空格，比如 @property (nonatomic, copy)，括号中分割属性的逗号后面必须有一个空格。

内存泄露

  • Block类型的属性，必须用 @weakify。

  • ReactiveCocoa，必须有 @weakify。

  • xib 和 storyboard 绑定的变量，用 weak。
